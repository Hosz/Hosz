## 🎮 Main Quest: Kyofuse

<div align="center">

### [Kyofuse](https://github.com/Hosz/Kyofuse)

**A CS2 social platform for competitive players, squads, team invites and community interaction.**

</div>

Kyofuse is my current main project: a full stack platform designed for **Counter-Strike 2 players** to create gamer profiles, publish posts, interact through comments and reactions, find squads, manage teams, and evolve inside a competitive community.

The project started as a personal portfolio project, but the idea is to keep developing it as something that could eventually go live and be used by real players.

### What I'm practicing with Kyofuse

- Modular monolith structure
- Back-end feature development with clear module boundaries
- Authentication and protected resources
- Feed, posts, comments and reaction flows
- Team and squad management rules
- Database modeling for social and gaming features
- API design for real use cases

- Project organization using issues, phases and planned roadmap
